# Digital Business Card

A digital business card built with **React** and **CSS**. This app showcases personal information, interests, and social links in a clean, interactive UI.

## Features

- **Personal Info:**
  - Display name, role, and brief introduction.
- **Interactive Sections:**
  - About, Interests, and Info components to organize content.
- **Footer with Social Links:**
  - Icons linking to social media profiles.
- **Responsive Design:**
  - Works well on desktops and tablets.

## How It Works

1. Open the app in your browser.
2. View personal information displayed in the main card.
3. Navigate through sections like **About** and **Interests**.
4. Click social media icons in the footer to visit links.

## Technologies Used

- **React** – For building UI components.
- **Vite** – Fast development build tool.
- **CSS** – Styling and layout.
- **JavaScript (ES6+)** – Logic and interactivity.

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/digital-business-card.git
```

2. Navigate to the project folder

```bash
cd dbca
```

3. Install dependencies & start the development server

```bash
npm install
npm run dev
```

4. Open the app in your browser
