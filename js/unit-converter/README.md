# Unit Conversion App

A simple and interactive web application to convert units of _Length_, _Volume_, and _Mass_ between **Metric** and **Imperial Unit** systems. The app automatically adjusts input font sizes for better visibility when entering large numbers.

## Features

- **Unit Conversion:**
  - Length: Meter ↔ Feet
  - Volume: Liter ↔ Gallon
  - Mass: Kilogram ↔ Pound
- **Interactive UI:**
  - Click on a unit category to update labels and conversion info.
  - Automatically shrink input text to fit within input boxes.
- **Real-time Conversion:**
  - Enter a value in one field, leave the other empty, and click convert.
- **Responsive Design:**
  - Works well on desktops and tablets.

## How It Works

1. Click on one of the categories in the bottom section (Length, Volume, Mass).
2. Enter a value in either the Metric or Imperial input field.
3. Click Convert.
4. The app will calculate the corresponding value and display it in the other input field.
5. The input font size will automatically shrink if the number is too large.
